Badgr
=====

An extension of the Badgr plugin for Atlassian Stash.
The original project can be found here:
https://bitbucket.org/stefankohler/stash-badgr/overview

Fixed to support Stash 3.0!

Added achievements:
- C++ levels.
- HTML files.
- Jade files.
- Stylus files.
- XO coder.
- Stom.
- Ohhhh no.
